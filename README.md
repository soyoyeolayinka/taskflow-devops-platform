# Production CI/CD Fullstack Application

## Overview

This project demonstrates a production-grade DevOps workflow for deploying and managing a containerized fullstack application.

## Tech Stack

- AWS EC2
- Docker
- Docker Compose
- Git
- GitHub Actions
- Nginx
- Flask
- React

## Architecture

User
→ Nginx
→ React Frontend
→ Flask API
→ PostgreSQL

## Goals

- Infrastructure provisioning
- Containerization
- CI/CD automation
- Monitoring
- Security
- Production deployment
